RECOMMENDATIONS FOR IMPROVED DISCOVERY:
================================================================================

1. Use browser-based scraping (Playwright) for JavaScript-rendered content
2. Check HackerOne/Bugcrowd APIs directly for company listings
3. Search for programs in:
   - robots.txt
   - sitemap.xml
   - HTML meta tags
   - JSON-LD structured data
4. Look for common security page patterns:
   - /security
   - /responsible-disclosure
   - /vulnerability-disclosure
   - /bug-bounty
   - /bugbounty
   - /disclosure
   - /.well-known/security
5. Extract contact info from:
   - security.txt Contact: field
   - Email addresses in page content
   - Links to security@domain.com
   - Links to HackerOne/Bugcrowd profiles
